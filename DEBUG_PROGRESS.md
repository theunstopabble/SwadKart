## Key Decisions
- **Route guard page flash**: Changed all 4 route guards (`PrivateRoute`, `AdminRoute`, `RestaurantRoute`, `DeliveryRoute`) to return `null` instead of `<PageLoader />` when `loading === true` ŌĆö prevents white page flash on auth check
- **Socket cleanup**: Added `cleanupRan` flag and moved `socket.off` into cleanup function; also added `socket.emit("leaveOrder")` on LiveTrackingMap unmount
- **Inventory bulk restock**: Replaced read-then-write with atomic `findOneAndUpdate` using `$inc` operator ŌĆö prevents race condition when admin and owner update simultaneously
- **Chat anonymous early-exit**: Anonymous callers now skip all 3 DB queries (products, lastOrder, user) and go straight to a lightweight product-only query
- **Biometric service**: Added `AbortController` + 10s timeout to all axios calls to prevent hanging on slow server cold starts
- **OrderAgain cancellation**: Added `cancelled` flag to useEffect cleanup to prevent setState on unmounted component
- **Firebase admin check**: Added `admin.apps.length === 0` guard in notificationController before using `getMessaging()`
- **Gamification update**: Restructured `findByIdAndUpdate` to use conditional `update.$push` ŌĆö prevents mixing `$push` operators with regular fields
